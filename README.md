# Apartment Booking System

Next.js + Prisma starter for a builder-grade apartment and shop booking system.

## Quick Start
1. Copy `.env.example` to `.env`
2. Set `DATABASE_URL` and `JWT_SECRET`
3. Install deps: `npm install`
4. Generate Prisma client: `npm run prisma:generate`
5. Push schema: `npm run prisma:push`
6. Seed data: `npm run prisma:seed`
7. Run app: `npm run dev`

## Default Login
- Email: `admin@builder.local`
- Password: `admin123`
